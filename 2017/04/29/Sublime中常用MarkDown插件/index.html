MarkDownPlunginsForSublime

2017-04-29

MarkDownPlungins

MarkDown插件

OmniMarkupPreview:实时在浏览器中预览，而MarkdownPreview是需要手动生成的和F5预览，如果有双屏的话，这个插件应该是个很不错的体验

快捷键如下：
  • Ctrl+Alt+O: Preview Markup in Browser
  • Ctrl+Alt+X: Export Markup as HTML.
  • Ctrl+Alt+C: Copy Markup as HTML.

在使用快捷键 Ctrl+Alt+O的时候很有可能遇到以下问题

“Error: 404 Not Found
Sorry, the requested URL ‘http://127.0.0.1:51004/view/29‘ caused an error:
‘buffer_id(这里会出现一个数字) is not valid (closed or unsupported file format)’
NOTE: If you run multiple instances of Sublime Text, you may want to adjust
the server_port option in order to get this plugin work again.”

这里给出一种比较给力的解决方案：
找到你刚才安装的OmniMarkupPreview的路径 …\OmniMarkupPreviewer/OmniMarkupLib/Renderers/libs/mdx_strikeout.py” 找到这个文件，打开它，将里面的makeExtension()方法代码用下面的代码换掉
def makeExtension(*args, **kwargs):
    return StrikeoutExtension(*args, **kwargs)
然后保存、关闭文件、并退出sublime，重现打开sublime即可。
